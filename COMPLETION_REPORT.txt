تم بنجاح تطوير نظام متقدم وشامل لحساب نسبة الامتثال (Compliance Score) بين
متطلبات المشاريع (RFP) والعروض الفنية (Technical Offers) باستخدام:

  • الذكاء الاصطناعي (Gemini 2.5 Pro)
  • البحث الدلالي المتقدم (Embeddings)
  • قاعدة بيانات Qdrant
  • تحليل ذكي ومتعدد الأوزان

النتيجة: نسبة امتثال دقيقة وموثوقة + تقرير مفصل يوضح الفجوات

================================================================================
                            🎯 الميزات الرئيسية
================================================================================

✨ استخراج ذكي للمتطلبات
   • استخدام Gemini AI للفهم الذكي
   • تقسيم تلقائي إلى 7 أقسام رئيسية
   • معالجة نصوص طويلة (حتى 15,000 حرف)
   • تصنيف هرمي ومنظم

✨ بحث دلالي متقدم
   • تحويل النصوص إلى Embeddings (384 بُعد)
   • حساب التشابه الكوسيني (Cosine Similarity)
   • 3 مستويات تطابق (عالي 0.75، متوسط 0.55، منخفض 0.35)
   • تكامل كامل مع Qdrant Vector Database

✨ حساب نسب دقيقة
   • نسبة التغطية (Coverage Ratio)
   • نسبة الجودة (Quality Ratio)
   • صيغة موحدة: (Coverage × 60%) + (Quality × 40%)
   • أوزان مختلفة لكل قسم
   • نسبة نهائية موثوقة (0-100)

✨ تقارير شاملة
   • النسبة الشاملة + نسب لكل قسم
   • نسب التغطية لكل قسم
   • أمثلة من الأجزاء المطابقة
   • قائمة بالمتطلبات الناقصة
   • مستويات الأهمية للمتطلبات الناقصة

================================================================================
                         💻 طرق الاستخدام
================================================================================

عبر FastAPI/HTTP:
───────────────────────────────────────────────────────────────────────────────
  curl -X POST "http://localhost:8000/api/compliance-score" \
    -H "Content-Type: application/json" \
    -d '{
      "source_text": "محتوى RFP...",
      "output_text": "محتوى العرض الفني..."
    }'

من داخل الكود Python:
───────────────────────────────────────────────────────────────────────────────
  from Backend.compliance import calculate_compliance_score

  result = calculate_compliance_score(
    client=client,
    model_name="gemini-2.5-pro",
    input_text=rfp_text,
    output_text=offer_text,
    embedder=embedder,
    qdrant_client=qdrant,
    use_advanced=True,
  )

  print(f"Score: {result['score']}")
  print(f"Categories: {result['category_scores']}")
